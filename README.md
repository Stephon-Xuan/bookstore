> 功能解释

首页主要是导航

1. 搜索

   点击进入[搜索页](#搜索)

2. 轮播

   广告书籍播放

3. 图书捐赠

   点击进入图书捐赠

4. 二手旧书

   点击进入二手书页面

5. 最新发布

   点击进入最新发布

6. 热门推荐

   点击进入热门推荐
